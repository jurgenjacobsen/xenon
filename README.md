# Xenon
Xenon transforms mundane chores into high-stakes side quests. Designed as a sister app to Nexus, it turns daily tasks into a shared RPG experience for modern homes.

- Quest-Based Chores: Tasks as bounties with XP rewards.
- Household Leaderboards: Compete for the MVP title.
- Real-time Sync: Powered by the same connectivity philosophy as Nexus.
- Modern Stack: Lightweight, fast, and built for the "smart home" lifestyle.
